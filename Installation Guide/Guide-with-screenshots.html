(A) Header

Header contains the Menu, the Search Box, blog Title and Description and Social Media Icons. We will go through each of them to make it easier for you to customize them the way you want. Go to Layout in your blog Dashbaord and you will see all the widgets present in this blog. In this section we will focus on the Blog's Header widgets.



Menu

Let's start with menu. Follow the guide to add/remove and customize menu items.

To add Menu in your blog. Click on the edit link in your Menu Widget, a new screen will appear with Title and Content Boxes.

You have to add the below code in the content box and then click on Save.

<span class='smenu'><a href="/">Home</a></span>
<div id="dd" class="wrapper-dropdown-3" tabindex="1">
    <span>Categories<i class='fa fa-angle-down'/></i></span>
    <ul class="dropdown">
        <li><a href="#">Nature</a></li>
        <li><a href="#">Sunset</a></li>
        <li><a href="#">Art Work</a></li>
        <li><a href="#">Models</a></li>
        <li><a href="#">Animals</a></li>
         <li><a href="#">Quotes</a></li>
         <li><a href="#">Kites</a></li>
         <li><a href="#">Fashion</a></li>
    </ul>
</div>
<span class='smenu'><a href="#">About</a></span>
<span class='smenu'><a href="#">Portfolio</a></span>


The Green color lines indicate that these are codes for normal menu link. These are simple links that don't open any dropdown menu. The red color lines indicate the codes for dropdown menu that appears when the categories link is clicked. You can add/remove these lines as you wish.

You have to replace # with the label url or whatever link you want. For example, you want Sunset text to have this link - http://myblog.blogspot.com/search/label/sunset
Your new code will be - <li><a href="http://myblog.blogspot.com/search/label/sunset">Sunset</a></li>

Search Box

To add the Search Box in your blog, simply click the edit link on your Search widget. A new screen will appear with Title and Content box. Paste the below code in the content box and click on the Save Button. You don't have to make any changes here.

<div id="sb-search" class="sb-search " >
    <form target="_top" method="get" id="search-form" action="/search">
        <input class="sb-search-input " onkeyup="buttonUp();" placeholder="Search ..." onblur="monkey();" type="search" value="" name="q" id="search" />
        <input class="sb-search-submit" type="submit" value="" />
        <span class="sb-icon-search"><i class="fa fa-search"></i></span>
    </form>
</div>


Blog Title and Description

To change your blog's title and description, click on the Header widget and simply change your blog's title and description and click on Save button. Go to your blog Dashboard and then clik on Settings. Here you can change your blog's title and description. You can also changes the colors and fonts used in blog's title and description from template designer. We will learn how to do that at the end of this guide.

Social Media Icons and Links

For adding social media links below your blog title and description. Go to Template, then click on Edit HTML. A new screen will appear with some codings in it. Press Ctrl+F on your keyboard and you will see a search box appear on the top right corner of your coding box. Write "Social Media Icons" in it without quotes and press Enter. It will search for social media icons in the template and take you there. Like this.


This is the code you will see.

<div class='pou'>
<a href='#' target='_blank'><img src='http://2.bp.blogspot.com/-SxvxnkCl4sA/VBs4YO_uaqI/AAAAAAAACGg/cdKWJhWvnkk/s1600/rss-64.png'/></a>
<a href='#' target='_blank'><img src='http://2.bp.blogspot.com/-7Ue5raG5lmU/VBs4YxaHNII/AAAAAAAACGk/yxcryBLfQkc/s1600/twitter-64.png'/></a>
<a href='#' target='_blank'><img src='http://4.bp.blogspot.com/-1Hmmo0f_vDE/VBs4W6m2EPI/AAAAAAAACGE/syrev4IzVMg/s1600/facebook-64.png'/></a>
<a href='#' target='_blank'><img src='http://2.bp.blogspot.com/-IkJR0rS2Hy8/VBs4XLUjzOI/AAAAAAAACGI/8mIduOPLqjE/s1600/instagram-64.png'/></a>
</div>

Now, you just have to replace # with the respective links. For example, your twitter username url is http://twitter.com/John . So after replacing # with http://twitter.com/John, it will look like

<a href='http://twitter.com/John' target='_blank'><img src='http://2.bp.blogspot.com/-7Ue5raG5lmU/VBs4YxaHNII/AAAAAAAACGk/yxcryBLfQkc/s1600/twitter-64.png'/></a>

The skyblue color code show the code used for one social media icon and link. To remove a social media item, simply delete the repective line. To add more social media icon, simply add more of these lines between the green code.

Like this -


STEP 2

Go to Template, then click on Edit HTML. A new screen will appear with some codings in it. Press Ctrl+F on your keyboard and you will see a search box appear on the top right corner of your coding box. Write "Instagram FEED" in it without quotes and press Enter. It will search for Instagram FEED in the template and take you there. Like this image. (click on the image to inlarge)

The code that we will see is this -
//<![CDATA[
$('.instagram-feed').socialstream({
    socialnetwork: 'instagram',

Here, you just have to change the red color text with your Instagram Username and green color number with the number of images you want to show on this widget.

For Instagram users -

https://instagram.com/oauth/authorize/?client_id=[CLIENT_ID_HERE]&redirect_uri=http://localhost&response_type=token

Try this -

https://instagram.com/oauth/authorize/?client_id=[CLIENT_ID]&redirect_uri=http://localhost&response_type=token&scope=basic+public_content+follower_list+comments+relationships+likes

Replace [CLIENT_ID] with the client id you generated on instagram developer account.

(D) Template Designer
Back To Top

I did my best to create a nice looking color scheme for you. But you might want to change some colors and fonts according to your needs. With this template, you can easily change the colors by selecting the element and picking up a color from the color wheel. You can do this with the help of blogger's Template designer feature. This Template is designed to support this cool blogger feature.

To access Blogger Template Designer, go to your blog dashboard, then click on template, a new screen will come with your template preview image and a Customize link below it, click on customize and your Template Designer will open up. Now click on Advanced in your Template Designer and you will see the list of all the section that you can customize. Click on them to customize their color and fonts.

This template supports color changing of not only the text, background and links but also the icons. You can easily change date, comment, navicon and backtotop icons color with the help of template designer. This will make your icons blend well with your new color scheme.
